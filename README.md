# 🐍 Python Calculator Portfolio

Aplikasi kalkulator modern berbasis **Python (PyScript)** yang dapat berjalan langsung di browser dan dipublikasikan menggunakan **GitHub Pages**.

![Python Calculator](assets/logo.svg)

## ✨ Features
- 🏠 Modern Landing Page
- 🧮 Calculator dengan operasi matematika dasar
- 🐍 Python Logic menggunakan PyScript
- 🎨 Responsive Glassmorphism UI
- 📱 Support Desktop & Mobile
- ⚡ Berjalan langsung di browser
- 🚀 Siap deploy ke GitHub Pages

## 🛠️ Technology
Project ini dibuat menggunakan:
- HTML5
- CSS3
- Python
- PyScript
- GitHub Pages
